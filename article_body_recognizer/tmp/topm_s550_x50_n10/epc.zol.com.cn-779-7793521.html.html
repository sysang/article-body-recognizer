【有料评测】惠普大师本ZBook Studio G8：用澎湃性能捕捉万千创意

2021-12-10 05:50:00 [ 中关村在线 原创 ] 作者：暗中观察的云鹏

当下，新媒体与智能制造行业飞速发展，工程师、设计师、数据分析师、视频剪辑师纷纷成为了时下的热门职业。新的工作需求也对他们手中的生产力工具提出了更高的要求，一台优秀的移动工作站成了以上专业人士不可或缺的工作伙伴。

提起移动工作站，人们往往有着又大又厚又沉的刻板印象，虽然不像商务本和游戏本一样被用户广泛了解和熟知，但它其实在市场中同样面临着激烈的竞争，各大厂商纷纷把自己尖端的技术、高档精致的做工以及扎实的堆料应用其中。

正所谓“闻道有先后，术业有专攻”，让专业的人做专业的事，同理专业的活就该交给专业的移动工作站来干。

这也是我们今天的主角惠普大师本ZBook Studio G8在当下产品种类繁多的市场上的核心竞争力所在。那么，作为惠普大师本ZBook Studio系列中的最新成员，它如何保障专业人士优雅运行各项工作负载？又是否担当得起官方赋予它的“大师”之名？相信看完下面的评测，你的心中一定会给出答案。

惠普大师本ZBook Studio G8

首先来看一看本次拿到机器的主要配置。

CPU：Intel酷睿i7-11850H

显卡：NVIDIA RTX A4000 Laptop

内存：32GB 双通道 DDR4 3200MHz

硬盘：KIOXIA 1TB PCIe3.0 SSD

外观方面，或许是因为上一代G7的模具足够优秀，本次G8并没有更换外观设计。依然是由铝合金打造，采用CNC一体成型工艺的机身，质感十足。各个边角都做了精细打磨，丝毫没有割手感。并且深灰色的金属外壳，不易沾染指纹，易于打理。

A面采用了时下流行的极简设计，正中央一枚ZBook系列标志性的字母“Z”Logo，增加了辨识度，也颇具时尚感。

其余接口、键盘以及机身厚度等部分由于模具没有变，因此本次不多做赘述，详细可以移步查看之前惠普大师本ZBook Studio G7的评测中的相关部分，总之还是保持了上一代的高水准，键盘舒适，接口齐全。

不过，此次惠普大师本ZBook Studio G8虽然是“旧瓶装新酒”式的更新，但也别急着喷，因为这次瓶子里的酒实在是太香了。首先是CPU部分，更新了英特尔最新的11代酷睿移动标压处理器，型号为i7-11850H。（CPU-Z此处识别名字有误）

对英特尔今年的产品线更新有一定了解的朋友都知道，这代英特尔在移动端上的更新远比桌面端来的更大，全线采用了10nm工艺制程，并配备了更大的三级缓存。那么反映到实际表现中究竟如何呢？我们可以先来看看CPU-Z中的对比测试。

从CPU-Z内置的数据上来看，即便是与上一代桌面端的i7-10700相比，i7-11850H单核领先了约13%，多核也有5%左右的提升，这还是建立在后者拥有更高主频和功耗的情况下。新的Tiger Lake架构和10nm工艺带来的提升肉眼可见。

如果与上一代的移动端CPU相比，这一差距会进一步加大，同样在CPU-Z中，惠普大师本ZBook Studio G8搭载的i7-11850H相比上一代移动端顶级型号i9-10885H，单核得分领先约22%，多核领先约16%。

左：i7-11850H 右：i9-10885H

在这样巨大的提升下，惠普大师本ZBook Studio G8的散热和稳定性可谓不负众望，得益于内部的液晶聚合物风扇和均热板设计，给全新的CPU带来了很大的发挥空间。在AIDA 64的烤机测试中，机器30分钟满载运行，CPU功耗保持在了60W左右，频率维持在3.2GHz，温度90℃全程没有出现降频。

并且，在这样的高负载场景中，整机的散热系统运转十分安静，没有恼人的噪音，并且主要键盘输入区域和掌托部分也没有烫手的情况出现。

显卡部分同样提升巨大，首先这款机器搭载的是一块英伟达RTX A4000移动版专业显卡，采用与最新的30系显卡相同的安培架构中的GA104核心，8nm工艺，5120个CUDA核心，8GB GDDR6显存，显存位宽256bit，对应消费级的3070系列。

其实A4000并不是什么新系列，其实就是由英伟达此前一直在做的Quadro系列更名而来，依然是面向制图、建模等用途的专业显卡产品线。

由于这一代的英伟达移动端显卡性能与功耗直接挂钩，因此显卡的最大功耗成了许多用户最关心的数据之一。

通过控制面板可以看到惠普大师本ZBook Studio G8上的这块RTX A4000显卡最大功耗被限制在了85W，在轻薄和性能上做出了一定程度上的平衡。

传统的DX12性能上，这款RTX A4000得分为8277，表现在这一规格的显卡里中规中矩，但整体属于市面上的中高端水准。

不过回到专业场景中，这块专业显卡与普通消费级显卡的差距就逐渐显露出来了，首先是V-RAY中的GPU测试，得分为1026。通过V-Ray官方排行榜可以看到，这在目前的移动端硬件中已经属于第一梯队的水平，非常能打。

然后是专业的SPECviewPerf 2020，SPECview是一个专业级、符合工业标准的OpenGL图形显示卡效能测试分析软件。在3dsmax、maya这些主流专业软件中，运行非常流畅。详细得分如下图。

这也是专业显卡的价值所在，手头有30系显卡的小伙伴可以挑战一下，即便是3080、3090来了，也不可能在这些项目中全面战胜这块规格仅相当于残血版移动端3070的RTX A4000。这是因为专业显卡在图形接口、指令集方面对专业软件的特殊优化和驱动相结合。这也是为什么在文章开头说出了那样一段观点，在实际体验中，对于有专业软件使用需求的用户而言，使用像惠普大师本ZBook Studio G8这样搭载专业显卡的工作站才是唯一选择。

在CPU和显卡的双重升级下，SPECviewPerf 2020的测试结果显示惠普大师本ZBook Studio G8在诸如maya和snx这些软件中的得分上代同规格CPU+专业显卡的2-3倍(战况太过惨烈，就不放图了)，其余各项软件也均有50%以上的提升。

此外，一块高速固态硬盘对于使用体验的提升也是巨大的。此次测试的机器搭载了一块来自铠侠的1TB PCIe 3.0固态硬盘，CrystalDisk读写速度均超过了3000MB/s，AS SSD Benchmark得分高达3685，十分优秀。这将大大减少专业软件读取体积较大的图纸、模型时的等待时间，提高用户的工作效率。

最后在PCMark 10整机测试中，这款机器整体得分6548分，同样十分优秀。

总结

作为一款专业向的细分领域产品，惠普大师本ZBook Studio G8在升级了最新的硬件配置后，整体性能较上代产品大幅提升，同时在模具上保留了诸如均热板、三面出风、大面积防滑脚贴等优秀设计，为用户带来了优秀的使用体验，符合其偏向中高端的产品定位。

惠普大师本ZBook Studio G8就像是一台阿斯顿马丁这类优雅的跑车。虽然远看外观相对低调，但近看却处处透着精致和时尚，并且也拥有着澎湃的动力。随着创意设计行业和互联网的深度融合，越来越多的创意工作者都有了移动办公的需求，对于他们而言，一台专业移动工作站既满
