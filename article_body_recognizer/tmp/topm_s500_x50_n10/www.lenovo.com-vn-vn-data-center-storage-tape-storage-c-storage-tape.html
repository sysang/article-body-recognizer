COST-EFFECTIVE DATA PROTECTION ARCHIVE SOLUTIONS

Tape technology is one of the most energy-efficient, secure, and portable long-term data storage solutions available. Lenovo offers a wide selection of tape storage products to meet your needs.

  1. TS4300 Tape Library

    The TS4300 is a high-density, highly scalable, easy-to-manage solution designed to keep data securely stored long-term, while reducing costs.

    Datasheet Product Guide
    Tìm hiểu thêm

    Thông số sản phẩm

    • Scale vertically with up to seven modules
    • Supports the latest Linear Tape-Open (LTO) 8 technology as well as generation 7, 6, 5 and 4
    • Up to 40 slots of LTO cartridge capacity per module
    • Up to 272 total slots with expansion
    • Outstanding capacity, performance reliability
    • Remote management through a standard web interface

  2. TS2900 Tape Autoloader

    The IBM TS2900 Tape Autoloader is a single-drive, low-profile automated tape solution and the first entry automation solution for small to midsize tape environments.

    Datasheet Product Guide
    Tìm hiểu thêm

    Thông số sản phẩm

    • Slim profile, 1U system designed for rack system environments for automated, high-capacity tape storage
    • Features the latest generation of Linear Tape-Open (LTO) with Ultrium 8 half-height technology designed for reliable performance in small to midsize open-system environments
    • Lowest entry price of any IBM tape automation offering half-height tape technology

  3. TS2280 Tape Drive

    Energy-efficient 8th generation tape drive suited for handling backup, save, restore, as well as archival data storage needs with high-capacity high data-transfer rates.

    Datasheet Product Guide
    Tìm hiểu thêm

    Thông số sản phẩm

    • Enables easy access to data stored on Linear Tape Open (LTO) Ultrium 8 cartridges through the use of IBM® Linear Tape File System technology
    • Supports media partitioning on LTO Ultrium 8 media Linear Tape File System
    • Yields double the compressed cartridge capacity

  4. TS2270 Tape Drive

    High-capacity 7th generation tape storage solution for businesses requiring a backup low-cost archival storage solution.

    Datasheet Product Guide
    Tìm hiểu thêm

    Thông số sản phẩm

    • Enables easy access to data stored on Linear Tape-Open (LTO) Ultrium 7, 6 5 cartridges through the use of IBM® Spectrum Archive™ technology
    • Supports media partitioning on LTO Ultrium 7 media
    • Yields more than double the compressed cartridge capacity with 87 percent better performance than LTO Ultrium generation 6 drives

  5. TS2260 Tape Drive

    Energy-efficient 6th generation tape drive suited for handling backup, save, restore, as well as archival data storage needs with high-capacity high data-transfer rates.

    Datasheet Product Guide
    Tìm hiểu thêm

    Thông số sản phẩm

    • Enables easy access to data stored on Linear Tape-Open (LTO) Ultrium 6 5 cartridges through the use of IBM® Linear Tape File System technology
    • Supports media partitioning on LTO Ultrium 6 media Linear Tape File System
    • Yields double the compressed cartridge capacity with more than 40% better performance than LTO Ultrium generation 5 drives
