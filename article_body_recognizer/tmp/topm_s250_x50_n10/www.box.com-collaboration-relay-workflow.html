Key features

Unlimited runs

Create and run an unlimited number of Relay workflows.

Event-triggered actions

Trigger a Relay workflow based on a regular schedule, file/folder/metadata/task events, or on demand.

Workflow templates

Get started quickly with pre-built departmental workflow templates and custom workflow templates.

Conditional logic

Leverage logic based on metadata attributes: date, dropdown, multi-select, and open text.

Flexible task assignment

Combine group tasks with any or all assignee requirements.

Automatic notifications

Push notifications are available across mobile and web.

Workflow management

Easily transfer workflows between users, and admins get user access control.

Activity tracking

Track progress of launched workflows, and view an audit history of in-progress and completed tasks.

Workflow trigger APIs

Kick off Relay workflows from external applications.
