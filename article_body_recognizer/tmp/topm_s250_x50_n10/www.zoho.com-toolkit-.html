What can I do with Toolkit?

  • Look up domain details
  • Get registry details
  • Run domain checks
  • Analyze email headers
  • Perform reverse lookup
  • Check for open ports
 

Look up domain details

Just enter the name of the domain to view the DNS records (MX, TXT, CNAME, A, AAAA, NS or SOA) associated with that domain.

 

Get registry details

Need to know who a domain is registered to or the provider that hosts the domain? Toolkit can quickly look these up, as well as other things like the expiry date of a particular domain.

Run domain checks

There are some records that every domain will have configured, like MX, SPF, and DKIM for example. Check the configuration status of these records for any domain you have in mind.

Analyze email headers

Email headers can be complex, technical, and loaded with information. Our Message header analyzer presents just the information that you need in a clear, readable format. View message details, email trails, and the other headers in clearly separated tabs.

 

Perform reverse lookup

Want to know if there's a domain hosted from a specific IP address? Tooklit has a reverse lookup tool for just that. Enter an IP address and it'll lookup any domains hosted for you.

Check for open ports

Check whether a particular port for a host is open with our Port Check tool before you start configuring any host.

Frequently asked questions

  • What are the checks that I can perform using Toolkit?

    You can look up the various records and registration details of any domain. You can also perform basic domain checks, reverse lookups, check for open ports, and analyze email headers.

  • Do I need to create a Zoho account to access Toolkit?

    No, you can access Toolkit without logging into any account.

  • How much should I pay to use the Toolkit?

    Toolkit is absolutely free and can be used by anyone.
