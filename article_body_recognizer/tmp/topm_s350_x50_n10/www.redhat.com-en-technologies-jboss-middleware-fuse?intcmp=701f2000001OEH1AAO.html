Red Hat Fuse

Distributed, cloud-native integration platform

Red Hat Fuse, based on open source communities like Apache Camel and Apache ActiveMQ, is part of an agile integration solution. Its distributed approach allows teams to deploy integrated services where required. The API-centric, container-based architecture decouples services so they can be created, extended, and deployed independently.

Red Hat Integration Options

Hybrid environment, single solution

Red Hat Fuse can be deployed in different environments, so diverse users get the integrations solution they need. All deployment options interoperate for collaboration between integration experts, developers, and business users.

Red Hat Fuse

Used primarily by integration experts and application teams with IT expertise, Red Hat Fuse can be deployed on-premise in a standalone configuration.

With Red Hat Fuse, you can:

  • Deploy on-premise
  • Use the same technology foundation as Red Hat Fuse on OpenShift
What does IDC think? Read the analyst whitepaper

Red Hat Fuse on OpenShift

For users who want a simplified IT infrastructure, Fuse can be deployed on OpenShift where containers are automatically provisioned and managed.

With Fuse on Openshift, you can:

  • Deploy container-native integrations easily and scale rapidly
  • Deploy in public or private clouds
  • Use the same technology foundation as Fuse Standalone
  • Use the Fuse Online browser based, drag-and-drop UI/UX to quickly create and deploy integrations. Learn more about Red Hat Fuse Online.

Red Hat AMQ

A Red Hat Fuse subscription includes Red Hat AMQ, a flexible, high-performance messaging platform.

With Red Hat AMQ, you can:

  • Connect anything, including the IoT, with a small IT footprint
  • Augment all Fuse deployments
Learn more about Red Hat AMQ

Red Hat 3scale API Management

Fuse connects anything, including APIs. With 3scale API Management you can make the most of APIs and benefit from decoupling services.

With 3scale API Management, you can:

  • Share, secure, manage, control, and monetize your APIs
  • Augment all Fuse deployments
Learn more about 3scale API Management
