Developer Tools

Red Hat CodeReady Workspaces

Red Hat® CodeReady Workspaces is a developer tool that makes cloud-native development practical for teams, using Kubernetes and containers to provide any member of the development or IT team with a consistent, preconfigured development environment. Developers can create code, build, and test in containers running on Red Hat OpenShift®. The user experience is as fast and familiar as an integrated developer environment (IDE) on their laptop.

Features and Benefits

Centralized, 1-click developer workspaces

Workspace configurations are centrally managed and easy to share using a devfile that specifies what developers on your team need to start working.

In-browser IDE

Includes an in-browser IDE, providing a desktop-like coding experience and allowing team members to update code from anywhere.

Air-gapped install

Red Hat CodeReady Workspaces can be downloaded and moved to run behind your firewall, isolated from the public internet.

Enterprise ready

Red Hat CodeReady Workspaces provides centralized monitoring dashboards and access control via your LDAP or Active Directory.

Red Hat OpenShift-native development

Development environments are containerized and running on the OpenShift Kubernetes platform.

Enterprise support

Red Hat’s award-winning support for your developer workspace.
