Backup, Email, Notifiche Web Push e CDN

Backup

La Macchina del Tempo, il nostro sistema di Backup, crea una copia di sicurezza (backup) dei contenuti del tuo blog - sia spazio web che database - in automatico e prima di un aggiornamento importante di WordPress gratis. Potrai riportare il tuo blog indietro nel tempo alla data in cui è stato eseguito il backup.

Backup automatico

Puoi pianificare la creazione automatica di una copia di sicurezza ogni giorno (con 7 backup precedenti ripristinabili) a 15,84 Euro/anno + iva, ogni settimana (con 2 backup precedenti ripristinabili) a 5,88 Euro/anno + iva o ogni mese (con 1 backup precedente ripristinabile) a 5,88 Euro/anno + iva.

Email

Casella email imap/pop3 con webmail - del tipo info@tuodominio.tld - con 5GB di spazio e numerosi alias a partire da 0,75 Euro/mese + iva. Acquistabile solo con dominio di secondo livello attivo.

Notifiche Web Push

Notifiche Web Push attivabili e disponibili gratis per mandare fino a 3 notifiche web push diretta sui dispositivi dei tuoi iscritti.

Servizio CDN

Oltre alla Million Page Cache, è possibile attivare gratis il servizio CDN di CloudFlare.

Domini aggiuntivi
