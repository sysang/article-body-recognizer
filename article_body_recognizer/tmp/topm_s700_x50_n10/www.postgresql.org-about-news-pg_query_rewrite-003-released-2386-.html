pg_query_rewrite 0.0.3 released

Posted on 2021-12-27 by Pierre Forstmann
Related Open Source

pg_query_rewrite is a PostgreSQL extension that can modify SQL statements received by PostgreSQL before the backend executes them. This tries to be similar to the Rewriter Query Rewrite Plugin in MySQL.

Release 0.0.3 of pg_query_rewrite was released. This is a maintenance release to add support to PostgreSQL 14.

  • Documentation: https://github.com/pierreforstmann/pg_query_rewrite/#readme
  • Download: https://pgxn.org/dist/pg_query_rewrite/0.0.3/
  • Support: use GitHub report tool at https://github.com/pierreforstmann/pg_query_rewrite/issues
