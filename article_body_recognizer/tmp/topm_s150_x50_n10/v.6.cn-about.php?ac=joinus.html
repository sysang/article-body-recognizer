PHP高级工程师

岗位要求：

1.精通 PHP，在不使用任何 framework 的前提下编写应用； 2.精通 MySQL 在 PHP 中的应用，包括基本查询，数据表优化，查询优化和 DB cache 引擎的配合； 3.精通 memcached 和 redis 的配置、使用、优化和除错； 4.有千万记录级 MySQL 表的实战经验； 5.4年以上 PHP 开发，有完整的非外包项目经验； 6.有REDIS 使用经验。

Android开发工程师

岗位要求：

1. 负责开发项目的核心功能，参与设计软件架构，优化 app 性能； 2. 追求极致的用户体验，善于雕刻和打磨产品； 3. 掌握 Rxjava,okhttp,MVVM,Fresco 组件化等等主流开源技术； 4. 熟练掌握 apm,oom,anr 等性能相关的优化； 5. 代码洁癖，对软件架构和设计模式热衷的喜爱； 6. 熟悉常用算法和数据结构； 7. 掌握 Flutter 跨平台研发技术，修改过 Flutter 渲染引擎加分。

直播产品经理

岗位职责：

1. 负责直播、社交类产品规划；根据市场需求，竞争态势和资源状况，手机、分析客户需求，合理规划产品版本开发计划，深入把 握产品核心价值，加强产品创新，确保产品功能满足用户的需求； 2. 定义产品功能；负责产品需求的管理，手机、提炼、整理客户需求，参与和知道编写产品功能定义文档； 3. 跟踪监控产品研发过程；及时了解产品的研发情况，跟踪监控产品研发的质量和进度，对产品测试进行管理，组织产品验收和发布； 4. 组织并跟进产品上线运营。与销售、研发、服务等多个部门进行充分沟通协调，定期向管理层汇报执行情况； 5. 负责编写产品基础销售工具，如产品介绍，技术白皮书，产品软件参数，产品宣传页； 6. 负责结合用户使用状况和客服反馈情况对产品进行改良和完善。

岗位要求：

1. 大学专科以上相关专业学历，2 年以上互联网行业同等职位工作经验； 2. 熟悉互联网产品设计流程，具备新产品敏感性，对用户群体有充分了解和把握； 3. 具有实际的项目管理经验，极强的沟通与协调能力； 4. 善于组织跨部门协作，并同时管理多个项目进程； 5. 具备良好的职业素养及团队合作精神，具有强烈的进取心，勤奋刻苦，学习适应创新能力强。

礼物特效师

职位描述：

1.负责直播类产品的礼物样式和礼物动效设计； 2.负责礼物活动相关的视觉设计 。

岗位要求：

1.熟练使用 AE、PS 等动效礼物设计软件； 2.有创意能力，可根据当前流行的礼物设计风格，设定自己的礼物风格； 3.有执行能力，能够灵活运用视觉和动效，达到产品所需的礼物效果，实操能力强； 4.有过 1 年或以上的动效礼物设计经验，沟通能力强； 5.本科及以上学历，动漫、美术及相关专业优先； 6.投递简历请务必附相关作品。

KA运营

岗位描述：

1.负责受理站内头部用户问题反馈，并提供优质解答服务，拥抱用户； 2.负责 KA 用户的反馈跟进，维护重要用户关系； 3.收集、汇总、分析用户建议和意见，输出报表及时向相关部门反馈用户需求及产品建议； 4.具有较强的沟通能力，可独立执行跨部门沟通任务； 5.对所负责的 KA 用户的各项数据指标负责。

岗位要求：

1.互联网直播用户侧相关经验者优先； 2.对 VIP 体系有所了解，能深度挖掘用户需求； 3.具有良好的沟通能力，自我驱动力。

内容运营

岗位职责：

1. 负责微信公众号、微博、视频号等新媒体的日常运营； 2. 从产品、内容、用户、推广等多个维度规划新媒体整体运营； 3. 负责直播平台社区运营工作，包括不限于社群内容的产出，社区活动的迭代与复盘，社区氛围营造，打造高粘性社区生态； 4. 定义和挖掘社区优质内容，培养社区核心用户，构建良好社区氛围。

岗位要求：

1. 本科及以上学历，传播学、新闻学、广告学、艺术学（编导、播音）等相关专业优先； 2. 有泛娱乐互联网公司 1-3 年新媒体或者社区工作经验，掌握 PS、PR、AE 等软件使用技巧，直播平台工作经验优先； 3. 有文案功底和数据分析能力，能及时从数据分析角度解决问题； 4. 具有较强的沟通协调能力，有责任心，注重细节，有较强的逻辑思维能力、创新能力、主观能动性强。

商务（KOL/KOC）

岗位职责：

1.负责拓展短视频平台等领域上符合产品类型相关的 KOL、KOC； 2.制定合作策略，负责全流程，协助完成创意策划、执行等全过程，提升渠道的拉新和活跃； 3.建立和完善产品用户画像数据，探索现有产品下的用户增长模式； 4.通过数据分析，量化评估运营效果，为运营策略提供优化方案； 5.对进行市场调研、对标同行业产品，推动业务的市场推广和商业转化； 6.结合行业发展方向和业务需求，探索与直播、短视频类机构新的合作模式。

岗位要求：

1.三年以上相关工作经验，熟悉短视频行业； 2.优秀的团队合作能力，工作主动意识强，较强逻辑思维能力； 3.具备较强的抗压能力和责任心，能高效协助各方共同完成目标。

优化师

岗位职责：

1. 负责操作今日头条、广点通、百度原生等信息流广告账户，根据 KPI 独立完成广告创建及优化工作； 2. 实时监控分析广告投放效果，制定投放策略，结合数据对广告投放策略进行调整，分析每日投放数据并跟踪转化效果及数据趋势，定期对在投渠道的效果进行整体分析，根据数据分析结果优化广告投放方案，持续精准化广告投放确保 ROI； 3. 针对产品及市场提出明确素材需求，制定创意及素材文案，及时跟进素材投放效果，定期对渠道效果进行分析总结，以数据为导向为产品制定推广策略，对所负责的账户具有清晰的定位和布局能力；收集用户及竞争产品相关市场信息及动态，并提出合理优化建议； 4. 熟悉各个信息流渠道人群属性和流量布局，对信息流素材产生有自己的见解，总结信息流方法论。

岗位要求：

1. 本科及以上学历，市场营销学专业优先； 2. 至少三年以上信息流类广告账户实操经验和指导经验，有自己的优化见解，负责账户年消耗千万级者同时具有直播、社交、短视频行业经验； 3. 思路清晰、活跃，有较好的学习能力，对数字敏感，擅长数据分析及归纳总结； 4. 有直播行业产品优化经验优先； 5. 责任心强，抗压力强，具有较好的团队合作精神和执行力，工作具有主动性同时能独立解决工作中遇到的问题。

证照管理

岗位职责：

1. 旗下各子分公司的各类资质证照的办理、变更等管理及维护； 2. 协助处理诉讼、仲裁、工商投诉等纠纷事务； 3. 负责档案制度完善（包括但不限于合同档案、诉讼档案及其他法律文件），对公司档案工作实施统筹管理； 4. 负责公司各类法律档案资料的收集、整理、保管、归档及销毁工作； 5. 领导交办的其他事宜。

岗位要求：

1. 全日制本科及以上学历，法学专业或者档案管理专业； 2. 两年以上相关工作经验，具备娱乐文化行业/互联网行业工作经验者优先；具备档案管理经验者优先； 3. 具备较强沟通能力和执行力； 4. 有团队协作意识。
