ASSOCIATION EDUCNAUTE-INFOS

Ce BLOG est une source d'information, pour tout ce qui touche de près ou de loin à la Pétanque surtout pour les Jeunes, dont le Challenge EDUCNAUTE JEUNES, les Educateurs, les Arbitres, les Règlements, etc. Vous y trouverez toutes les informations sur les ateliers bénéfiques pour les écoles de pétanque. Vous aurez des reportages sur les concours, les clubs, les écoles de pétanque, les diverses interviews sur les jeunes, les dirigeants, etc.. Vous avez aussi des "DIRECT" sur les championnats de France, Européen, et du Monde, avec, et leurs résultats. Le Blog n'est pas misogyne, il relate beaucoup d'infos sur la Pétanque féminine.
