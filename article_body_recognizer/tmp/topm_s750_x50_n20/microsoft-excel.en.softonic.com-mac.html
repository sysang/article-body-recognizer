Enjoy Excel's new features

Microsoft Excel is a productivity application from Microsoft. Part of the office suite bundle, this app is arguably the most famous and widely used spreadsheet software. It provides a wide array of capabilities, making it the go-to source for spreadsheet editing, macro programming, and much more.

Despite being already filled to the brim with helpful tools, Excel brings new innovative features. The 2021 version allows you to collaboratively work with others by adding co-authoring, dynamic arrays, Xlookup, and LET functions. It also made sharing easier, thanks to the integration with SkyDrive and SharePoint. However, this version does not support 64-bit.

Visual refresh and new functions

When it comes to spreadsheet editing, Microsoft Excel takes the reign. Known as the workbook gold standard, this app outmatches its competition with its wide selection of tools and advanced features. However, even though it is already packed with useful features, the latest update adds yet more functions that you can use. Not only that but the revamp also brings a slicker OS X style interface.

Users can now work with modernized Start and newly refreshed tabs in the ribbon, allowing easy access to the app's functions. It also provides right-click cell menus and powerful configuration tools in the main setting window. Moreover, the clean and clear style of the window also makes the app easier for the eyes. However, long-time users need not worry about adapting to the latest interface design. You only need to get accustomed to the newly introduced integration into the Office 365 suite.

More than a facelift, Excel 2021 also brings new functions. One of which is Xlookup. With this, users can find anything they need in a table or range, row by row. Unlike Vlookup, this function defaults to an exact match rather than an approximate. It can also perform horizontal and vertical lookups, replacing both Vlookup and Hlookup. Xmatch function, meanwhile, searches for a specified item in an array or range of cells. This also returns the item's relative position.

The latest version also makes calculation easier. Its dynamic arrays let you write one formula and return an array of values. It also expedites calculations and insights with six more functions, including filter, sort, sortby, unique, sequence, and randarry. Meanwhile, the Let functions assign names to calculation results, allowing storing intermediate calculations, values, or defining names inside a formula. Like variables in programming, this is accomplished through native formula syntax. 

Easier collaboration and better performance

Besides new tools and the introduction of the ribbon, Microsoft Excel also makes sharing easier, thanks to its integration with SkyDrive and SharePoint. Not only that, but you and your colleague can open and work on the same Excel workbook. With the co-authoring feature, you can see each other's changes quickly. The app also promotes better collaboration with modern comments. With this, you can have control whenever you send comments to your co-authors. It also makes commenting consistent in your workbooks and other Office apps.

Long-time users should be glad to know that all the old features of the app are still available. With this, you can still use the advanced filtering and analyzing of data. However, the performance has vastly improved, and the stability and speed are also enhanced across Excel. With this, you can get faster calculations from common functions. Moreover, the latest version is much lighter on the resources and hardly leaves any footprints. Although, as mentioned, it is only available in a 32-bit version.

Still the best workbook

Overall, Microsoft Excel is the best choice if you are looking for a workbook program that can cater to your needs. It is a powerful and versatile tool that is, by far, the best when it comes to spreadsheet creation and editing. And with the latest version, you have more functions at your disposal. The performance is also more stable without taking up more of your storage space.
