Tumblr Install Guide

Statcounter provides customisable hit counters, visitor tracking, web analytics and website stats for Tumblr.

  1. Sign up for an account at Statcounter to get your code.

    These instructions will be repeated after you've created an account and added your first project.

  2. Copy the StatCounter code.
  3. Log into your Tumblr account (opens in a new window) with your username and password.
  4. Go to the person icon at the top right of your screen select "Edit Appearance" from the dropdown menu.
  5. On the new page, select "Edit Theme".
  6. On the "Edit Theme" screen paste the code in the "Description" section check that "Show Description" is turned ON.
  7. Select Save Exit.
StatCounter is now active on your Tumblr site and will immediately begin recording visitor information. Note: Make sure your blog is set to public view, this enables your blog to be viewed by all (non tumblr users), settings for the same can be viewed in this youtube video

These instructions have been compiled and verified: November 2014
