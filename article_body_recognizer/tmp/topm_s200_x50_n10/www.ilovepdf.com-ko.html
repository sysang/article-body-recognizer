PDF 병합, PDF 나누기, PDF 압축, 오피스 파일에서 PDF로 변환, PDF에서 JPG로 변환, JPG에서 PDF로 변환을 위한 무료 온라인 PDF 도구. 설치 필요 없음

PDF 합치기

원하는대로 PDF 파일들을 병합하고 통합하세요. 매우 쉽고 빠릅니다!

PDF 분할

페이지 범위를 선택하고, 한 페이지를 분리하거나 문서의 각 페이지를 독립된 PDF 파일로 변환하세요.

PDF 압축

최고의 품질을 유지하면서 PDF 파일 크기를 줄이세요. 당신의 PDF 파일을 최적화하세요.

PDF Word로 변환

PDF 파일을 편집하기 쉬운 PPT 및 PPTX 슬라이드쇼로 쉽게 변환하세요. 변환된 WORD 문서는 100% 정확합니다.

PDF 파워포인트로 변환

PDF 파일을 편집하기 쉬운 PPT 및 PPTX 슬라이드쇼로 변환하세요.

PDF Excel로 변환

몇 초만에 PDF 데이터를 Excel 스프레드 시트로 가져올 수 있습니다.

Word PDF로 변환

DOC 및 DOCX 파일을 PDF로 쉽게 변환하세요.

파워포인트 PDF로 변환

PPT 및 PPTX 슬라이드쇼를 PDF로 쉽게 변환하세요.

Excel PDF로 변환

EXCEL 스프레드시트를 PDF로 쉽게 변환하세요.
새 기능!

PDF 편집

PDF 문서에 텍스트, 이미지, 도형 또는 주석을 추가하세요. 추가된 컨텐츠의 크기, 글꼴 및 색상을 편집할 수 있습니다.

PDF JPG 변환

PDF 페이지를 JPG로 변환하거나 PDF에 포함된 이미지를 쉽게 추출하세요.

JPG PDF 변환

이미지에서 PDF로 변환. 방향과 여백을 조절하세요.

페이지 번호

PDF 문서에 페이지 번호를 쉽게 추가하세요. 위치, 크기, 폰트 선택이 가능합니다.

워터마크

PDF에 이미지 또는 문자를 추가하세요. 글자, 투명도 및 위치를 선택할 수 있습니다.

PDF 회전

원하는대로 PDF를 회전하세요. 한 번에 여러 PDF 파일을 회전하세요!

HTML PDF 변환

HTML 웹 페이지를 PDF로 변환합니다. 원하는 페이지의 URL을 복사하여 붙여 넣은 다음 클릭하여 PDF로 변환합니다.

PDF 잠금해제

PDF 파일의 암호를 제거하여 필요에 따라 사용하세요.

PDF 보호

PDF 파일을 암호로 보호하십시오. 무단 액세스를 방지하려면 PDF 문서를 암호화 하십시오.

PDF 구성

원하는대로 PDF 파일의 페이지를 정렬하십시오. 편리하게 PDF 페이지를 제거하거나 문서에 추가하십시오.

PDF PDF/A로 변환

PDF를 PDF/A로 변환하여 ISO 표준으로 오랫동안 문서를 보존하세요. 나중에 PDF 파일을 열어도 서식이 유지됩니다.

PDF 복구

손상된 PDF를 복구하고 손상된 PDF에서 데이터를 복구하십시오. 수리 도구로 PDF 파일을 수정하십시오.
