For Free Software Activists…

Support current FSF campaigns

Please support the FSF campaigns, currently including: against DRM, Upgrade from Windows 8, PlayOGG, against software patents, OpenDocument, free BIOS, and against ACTA. The overall community site for the global movement for free software is LibrePlanet.

Help fight software patents

Please help end software patents worldwide. Please also support anti-software patent efforts in Europe specifically. Watch and share the movie Patent Absurdity.

Call on WIPO to change its name and mission

Please support this declaration calling on WIPO to change its name and mission.

Enhance the Free Software Directory

Contribute to the Free Software Directory: submit new packages, corrections to existing entries, and helping with maintenance.

Start a GNU/Linux User Group in your area, and send us your group's information

For more information: GNU Users Groups.

For Software Developers…

Contribute to high priority free software projects

As listed on the FSF's high-priority projects web page.

Contribute to GNUstep

Please contribute as a user and developer to GNUstep, a free object-oriented framework for application development, and help it achieve the status of a complete and featured desktop environment.

Take over an unmaintained GNU package

These GNU packages are looking for a maintainer: cfengine, freedink, freetalk, halifax, mcron, quickthreads. And these packages are looking for co-maintainers: aspell, bison, gnuae, gettext, gnubik, metaexchange, powerguru, xboard. See the package web pages for specific information, and this general information about GNU packages and maintenance, and then email maintainers@gnu.org if you have time and interest in taking over one of these projects.

Contribute to the development of a GNU package

See “help wanted” requests from the maintainers of many packages.

Make your program a GNU package

To make your free software package part of the GNU System, see this information on starting your application.
