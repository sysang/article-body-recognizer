Enterprise Linux and open source support services

Canonical provides 24/7, open source software support for the full stack through Ubuntu Advantage. Customers can choose between two support offerings - Ubuntu Advantage for Applications and Ubuntu Advantage for Infrastructure.

Ubuntu Advantage

Applications

Security and support for open source database, LMA, server and cloud-native apps.

  • Security patching for over 28,000 packages in Ubuntu Universe and Main repositories
  • Up to 10 years of Linux support for Ubuntu LTS releases
  • Technical support for open source database, LMA and cloud-native apps

Infrastructure

Security and IAAS support for open source infrastructure.

  • Security patching for more than 2,500 packages in Ubuntu Main repository (after 5 years of free LTS security patching)
  • Up to 10 years of Linux support for Ubuntu LTS releases
  • Technical support for OpenStack, KVM, MAAS
