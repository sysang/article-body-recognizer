Launchpad is a software collaboration platform that provides:

  • Bug tracking
  • Code hosting using Bazaar and Git
  • Code reviews
  • Ubuntu package building and hosting
  • Translations
  • Mailing lists
  • Answer tracking and FAQs
  • Specification tracking
