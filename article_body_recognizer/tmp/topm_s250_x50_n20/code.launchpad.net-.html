Code in branches

Launchpad can host your project’s source code using the Bazaar version control system. We also import over 2000 CVS, SVN and Git projects, so you can use Bazaar with those too. (Read our guide)

Launchpad can build Ubuntu packages directly from branches using recipes. We currently build over 100 different packages into PPAs automatically using this method. (Learn more about recipes)
