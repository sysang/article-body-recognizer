Registered 2017-02-02 by Bret Barker

The Snap Store supports software discovery and delivery for Snaps.

Note this project is for the backend services, not the desktop GUI client, see below.

Information about building and publishing snaps can be found at https://snapcraft.io/

Store operation status is tracked on http://status.snapcraft.io/

Discussion is in #snappy on Freenode and at https://forum.snapcraft.io

Snappy issues not directly related to the backend Store services should be filed at: - https://launchpad.net/snap-store for the Desktop Snap Store Client - https://launchpad.net/snapd for snapd specific bugs - https://launchpad.net/snapcraft for snapcraft specific bugs - https://launchpad.net/snapstore-snap for Snap Store Proxy bugs - https://launchpad.net/snappy for general bugs
