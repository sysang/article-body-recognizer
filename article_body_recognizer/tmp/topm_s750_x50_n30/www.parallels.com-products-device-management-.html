Extend Microsoft Endpoint Configuration Capabilities to macOS and iOS

Flexible Enrollment for Mac and Apple Mobile Devices

With Parallels Device Management there are multiple enrollment options for Mac and Apple mobile devices, including:

  • Active Directory System Discovery: Find and enroll the Mac computers and devices joined to the AD domain with ease.
  • Apple Device Enrollment Program: Zero-touch device enrollment, configuration, and a simplified setup experience.
  • User-Initiated MDM Enrollment: Enroll Mac, iPhone and iPad in MECM and MDM instantly using the enrollment URL.
  • All Apple Devices in Configuration Manager: Easily manage mobile devices, including configuring profiles, collections, reporting, lock and wipe.

Comprehensive Inventory

Enjoy unified endpoint management that has all types of devices in one place, and offers MECM administrators a coherent view on Windows, macOS, iOS and iPadOS device collections.

  • Inventory reports for Mac: Gather information and generate reports on hardware, applications, encryption status, user logons and custom inventory.
  • Inventory reports for Apple mobile devices: Generate reports on hardware and software important for security and accountability.
  • Software metering for Mac: A simple way to know the needs of employees and identify the risks for the company.
  • Custom inventory for Mac: Gather virtually any information for flexible asset management.

Security Compliance Enforcement

Parallels Device Management has a number of security and compliance protocols.

  • Lock or wipe managed Mac computers and Apple mobile devices remotely
  • Deploy Configuration Profiles to macOS and Apple mobile devices
  • Enjoy easy-to-use macOS/iOS configuration profile editor
  • Use MECM task sequences like you do with PCs
  • Enable FileVault 2 full-disc encryption
  • Leverage centralized FileVault 2 private keys management

Software Deployment and Patch Management

  • macOS software deployment via MECM packages and applications
  • macOS update management and reporting
  • Apple Volume Purchase Program (VPP) application deployment
  • Self-service application portal for Mac end users

Hardware and Software Management

  • MECM Software Metering for macOS applications
  • Hardware and software inventory reports for macOS and iOS
  • User logon information reports
