Frequently Asked Questions

How can I try or purchase Parallels Desktop for Chrome OS?

Please contact our Sales team at https://www.parallels.com/products/desktop/chrome/.

How can I get support if I have an issue?

Please contact your system administrator directly to resolve the issue with Parallels Desktop. If you are the system administrator, then you can contact support at Support Portal. If support options for Parallels Desktop for Chrome OS are not available to you at Support Portal, troubleshoot using KB 125132.

How does licensing work in Parallels Desktop for Chrome OS?

A license that you purchase covers a specific number of users and is valid for a specified time (1 year in most cases). Additional licenses may be purchased and added to your domain at any time, and users will migrate to new licenses seamlessly as previous licenses expire. To manage Parallels Desktop for Chrome OS licenses, use Google Admin console: https://support.google.com/chrome/a/answer/9681911?hl=en

How do I set up Parallels Desktop for Chrome OS?

If you are a Parallels Desktop for Chrome OS system administrator, see the Administrator Guide.

Where can I find more information about Parallels Desktop for Chrome OS features?

You can read about existing features in the User Guide.
