Install AddToAny in Joomla

AddToAny's official Joomla extension is available as a plugin module for Joomla 3.0+.

To install in Joomla 3.2+:

  • Go to Extensions > Manage > Install
  • Under Install from Web, search for AddToAny, then Install the extension
  • Enable AddToAny in Extensions > Plugins, then you can click the plugin's name to configure AddToAny

To install in Joomla manually:

  • Download AddToAny for Joomla
  • Go to Extensions > Manage > Install
  • Under Upload Package File, choose the ZIP file you downloaded, then click Upload Install
  • Enable AddToAny in Extensions > Plugins, then you can click the plugin's name to configure AddToAny
