Supported Browsers

We build Big Cartel to be used in the latest versions of modern browsers. That way, we can devote our time to working on cool new features instead of maintaining old stuff for ancient browser versions. RIP, Netscape Navigator.

Here’s the rundown of our supported browsers and minimum requirements:

  • Chrome 61+
  • Safari 10.1+
  • Firefox 63+
  • Edge 16+
  • Opera 12+

Check this site to make sure you’re up-to-date.

Using an older version? No worries and no judgment - updating is free and easy.

Heads up: Big Cartel may not work in beta versions of these browsers - be sure to download the latest commercial version for the best experience.
