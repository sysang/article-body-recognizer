Una manera divertida de explorar
Kids' Wish List es una forma sencilla y divertida para que tus hijos descubran ideas de regalo geniales y para guardar una lista con lo que les gusta. Ideal para niños de 3 a 11 años.
Un espacio seguro
Disfrutarán de un entorno seguro, donde solo se mostrarán productos adecuados para su grupo de edad y donde será necesaria tu contraseña para salir de la experiencia.
Conceder sus deseos
Cuando hayas terminado, la lista estará disponible en tu cuenta, donde podrás gestionarla y compartirla con amigos y familiares. Sencillo.
Para cualquier ocasión
Ya sea para un cumpleaños, para Navidad o para un regalo especial, Kids' Wish List te ayudará a adquirir el regalo perfecto para cada ocasión.
