Frequently asked questions

Elementor is the ultimate WordPress website builder that is both free and open-source. It enables you to create amazing WordPress websites intuitively, professionally, and without having to write code.

Elementor Pro offers cutting-edge tools and advanced capabilities web creators need to build professional pixel perfect websites.

It includes the industry-leading theme builder, design-oriented popup builder, visual form builder, popular marketing integrations, WooCommerce builder, dynamic content capability, custom fields ability, motion effects, premium support, and many other revolutionary features.

The free version of Elementor is available for download on the WordPress repository. Elementor Pro can be purchased directly from the Elementor website.

A Template Kit is a collection of pre-designed templates that help you create a full website for a specific industry, with the help of a website builder like Elementor.

Template Kits can give website creators lots of pre-designed templates for a great variety of websites - they can be used to complement your existing Theme or as an alternative to a Theme. The starter content is great quality and can be easily and reliably installed.

An Add On is a plugin designed to expand the functionality of your website, with the help of a website builder like Elementor.
