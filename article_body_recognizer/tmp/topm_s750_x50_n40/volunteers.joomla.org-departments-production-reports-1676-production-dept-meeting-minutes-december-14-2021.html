Production Dept. Meeting MINUTES - December 14, 2021

Participants: Benjamin Trenkle (Release Lead 4.1), David Jardin (Team Leader), George Wilson (Release Lead 4.0), Harald Leithner (Release Lead 5.0), Christiane Maier-Stadtherr (Team Leader), Viviana Menzel (Assistant Team Leader), Llewellyn van der Merwe (Assistant Department Coordinator), Niels Braczek (Team Leader), Sigrid Gramlinger (Team Leader), Shivam Rajput (Team Leader), Philip Walton (Marketing Liaison), Robert Deutz (Team Leader), Tobias Zulauf (Release Lead 3.10) Absent with notice: Marco Dings (Department Coordinator) Absent without notice: Jacob Waisner (Team Leader) , Mike Brandner (Team Leader), Hannes Papenberg (Team Leader)

Security Strike Team (David Jardin)

  • JSST received several requests about potential vulnerabilities in Joomla related to the log4j issue - but as log4j is based upon Java and Joomla is based upon PHP, there is no issue in the CMS
  • JSST did a review of the project’s infrastructure for vulnerable assets related to log4j without any findings.

Release Leads (Benjamin Trenkle, George Wilson, Tobias Zulauf, Harald Leithner)

* Release Lead 4.2 (Harald Leithner):

  • Resigned as Release Lead for Joomla! 4.2

* Release Lead 4.1 (Benjamin Trenkle):

  • Joomla! 4.1 Beta 1 released
  • Joomla! 4.1 Beta 2 will come at 21.12.2021

* Release Lead 4.0 (George Wilson):

  • Joomla 4.0.5 is out
  • Working on more deprecation warnings for PHP 8.1
  • Working on updated system test images with Harald

* Release Lead 3.10 (Tobias Zulauf):

  • Joomla 3.10.4 has been released
  • Joomla 3.10.5-rc1 is scheduled to be released on the 8/9th January 2022
  • Joomla 3.10.5 is scheduled to be released on the 18th January 2022

* Maintainer Framework (Niels Braczek):

  • Still working on the monorepo, I found some issues with the framework packages during the development; will soon start with fixing issues with PHP 8.1.

CMS Release Team (Sigrid Gramlinger)

  • Search for release leads is going on.
  • We have some testers, but not enough, we need more automated testing.
  • We will try to motivate more people for testing with direct postings to different language or regional communities (like Dutch or Australian).
  • We have new team members.

Finalized Motions

Motion Release Lead Joomla! 5.0: PROD2021/003 Elect Harald Leithner and Niels Braczek as Release Leads for Joomla! 5.0 Proposed by Benjamin Trenkle, seconded by Sigrid Gramlinger 11 in favour, 0 against, 0 abstain (4 unattended) PASSED

Change Joomla! Core to PSR-12: PROD2021/004: Update the code style to PSR12 as early as possible but latest for 5.0 Proposed by Benjamin Trenkle, seconded by Harald Leithner Moved to E-Mail vote Result [2021/12/20]: 9 in favour, 2 against, 2 abstain (2 unattended) PASSED
