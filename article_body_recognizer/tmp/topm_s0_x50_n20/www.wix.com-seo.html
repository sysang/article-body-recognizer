FAQ

Can I customize meta tags on my Wix site?

Yes. From the Wix Editor, open the SEO settings of any page and enter the text you want to appear in the relevant fields (i.e., title, description). You can also add custom meta tags and edit URL slugs for each page of your site. Once you’ve edited field values, your custom settings will override default content provided by Wix.

Can I edit my robots.txt file on Wix?

Yes. All Wix sites automatically come with their own writeable robots.txt file. This file tells search engines which of your site pages should/should not be indexed. For more control over crawl requests to your site, you can write your own directives in the Robots.txt File Editor. To view your site’s robots.txt file, add “/robots.txt” to your root domain (i.e., www.domain.com/robots.txt). Find out more

Can I set custom canonical tags on Wix?

Canonical tags tell search engines which URL is your primary version and consolidate URLs on your site that contain duplicate content. Additionally, canonical tags can be used to simplify tracking analytics. 

By default, Wix adds the page URL to the canonical tag. This is called a self-referencing URL because it makes reference to itself as the canonical. Since most of your site pages won’t have duplicate versions, self-referencing canonical tags are used—an SEO best practice. You can always customize the default canonical tag for any page. Learn more

What kind of page rendering is supported by Wix?

The SEO infrastructure of all Wix sites is based on server-side rendering (SSR) with crawl efficiency in mind. Additionally, every Wix site automatically includes a sitemap that gets submitted to Google. Wix maintains your sitemap, keeping up to date with all your site content. To view your sitemap, add sitemap.xml to the end of your root domain (i.e., www.domain.com/sitemap.xml).

Can I override default structured data provided by Wix?

Yes. With Velo’s structuredData API, you can modify and tell search engines which content you want to be eligible to appear as a rich result. Set your structured data with a list of structured data objects in the JSON-LD format according to schema.org. Keep in mind, the structured data you set will override any default information provided by Wix.

Because SEO data should be read by search engines, make sure to set this data in the onReady( ) event handler. Find out more

How can I optimize images on my Wix site for SEO?

Upload high-quality images to your site and Wix will automatically optimize them to the exact dimensions displayed on the screen—without any loss of quality. This includes converting images to WebP format for faster downloads. Additionally, Wix supports SEO-friendly formats, like JPG, and compresses SVGs. Find out more
