How to change internet browsers to Opera from Chrome, Firefox, or Edge

When you switch internet browsers to Opera, you will automatically be prompted to import whatever data you want from your old web browser. If you are already using Opera, you can import everything by following the steps below. Lose nothing, gain everything when you switch internet browsers.

Step 1

Download Opera Browser

First, simply download and install Opera Browser.

Download Opera

Step 2

Click the Opera logo

Launch Opera, click the Opera logo in the top-left corner and click Settings (Preferences on Mac).

Step 3

Import your data

Under Synchronization, click Import bookmarks and settings. You can choose which data to import, as well as which browser to get it from.

Step 4

Set your default browser

While in Settings, select Make Opera my default browser. Learn more about configuring Opera.
