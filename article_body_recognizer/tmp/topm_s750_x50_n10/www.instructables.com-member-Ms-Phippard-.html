  • Ms-Phippard commented on Ms-Phippard's instructable Sculpting Critters in Fusion 360

    In Fusion 360 tools menu, there is a "3D print" button. This will allow you to save an STL file. If your object is one piece, it will be one file. If you have multiple "bodies" you can save multiple STL files this way (such as if you wanted a dual-extrusion print), or you can combine the bodies first before saving. Using the 3D print button, you can also send directly to a connected 3D printer, if you have one set up.
